### 무엇을 작성하나요? ✍️

<br/>

관심 있는 분야에 관해서 공부한 뒤, 공부한 내용을 정리해 작성합니다. <br/>
가장 관심이 있는 프론트엔드 관련 기술 이외에도 흥미가 생기는 분야는 가리지 않고 작성합니다. <br/>

다른 개발 블로그를 참고하며 많은 도움을 받았던 경험으로, 누군가에게 도움이 되었으면 하는 마음에서 시작하게 되었습니다.
